Projects

Procect 1 : Room Temperare Detector
In this project, we made a Room temperature monitoring system using LM35 Temperature Sensor and Arduino. This system was able to measure room temperature continuously and show the value on a Display. The LM35 is a 3- terminal Temperature Sensor IC that was used to sense ambient and surface temperatures. It measured temperature in the range of -55°C to 150°C. The sensor provided an analog output voltage that was proportional to the temperature. So, it was a perfect sensor to measure Room temperature. We built a very small and portable Temperature Monitoring System/Device. The aim was to make it be easily used in offices, schools, buildings, and industries to measure temperature. We then created an android application to receive a notification on a personal cellphone to be notified about the temperature output. I contributed to the project by creating the front end development using HTML and CSS. The outcome of the project was 100 percent, we managed to solve all the problems we faced on the project.
Project 2 : Intelligent Agricultural System
The objective of this project was to develop an intelligent agricultural system leveraging ESP32 and Arduino technology, intergated with Firebase for data storage and web application for remote monitoring and control. By combining these technologies, i aimed to address several existing gaps and challenge in traditional agricultural practices, particularly in the areas of water management, environment monitoring, and decision-making support. The system was intended to be easily deployed and accessed by web browsers on deskop, laptop, and mobile devices. The ESP32 microcontroller, sensors, and relay module were among he hardware elements that was incorpareted ino the overall sysem architecture to guarantee dependable data acquisition and control capability.
